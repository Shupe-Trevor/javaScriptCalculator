Phase 0:Requirements Specification

For this assignment I need to write a Js- Calculator only using JavaScript and basically no HTML. This webpage will have a main div that will contain the expression building UI. This UI will have an input for the type of mathematical expression. There will be two number inputs for the Operands. There will be a color input as well as. There need to be a compute button that will create the expression inputted. Onclick of this button JavaScript will create a new Result div with the background color matching the input color from the user.

The new result divs will contain the time of the creation and the expression build including the answer. The result divs will have an onclick that will remove them from the page.

Phase 1: System Analysis

The input for this Calculator will be two number inputs, an math type input, and a color input. There inputs are dependent on the user.

The color and the math type will have default inputs, The numbers will not. Upon error a new red div will display the error message.

Phase 3: Implementation

var makeNewDiv = function(text, backColor, textColor) {

const date = new Date()
const [month, day, year] = [date.getMonth(), date.getDate(), date.getFullYear()];
const [hour, minutes, seconds] = [date.getHours(), date.getMinutes(), date.getSeconds()];
let divDate = document.createElement("span")
divDate.setAttribute("Style", "color:" + textColor)
divDate.setAttribute("Class", "timestamp")
divDate.textContent = month + "/" + day + "/" + year + " " + hour + ":" + minutes + ":" + seconds;
divDate.textColor = textColor;
let divText = document.createElement("span")
divText.setAttribute("Style", "color:" + textColor)
divText.textContent = text;
let newDiv = document.createElement("div")
newDiv.setAttribute("Style", "Background: " + backColor)
newDiv.setAttribute("Class", "result")
newDiv.appendChild(divDate)
newDiv.appendChild(divText)
document.body.insertBefore(newDiv, document.getElementsByClassName("result")[0]);
newDiv.onclick= () =>{
newDiv.remove()
}
return 0
}
